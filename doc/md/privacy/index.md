# Privacy Policy

No user data is gathered in the operation of this Add On. Sensitive data such as RSS feeds and Discord Webhooks are never stored beyond Google Sheets and remains as secure as the owners of such Spreadsheets intend.

The spreadsheet ID is used as an identifier for HTTP requests. Please ensure your spreadsheets are appropriately access-controlled as the URL may be deduced from this information. This is done to ensure tracability in the case of nefarious actors.

A non-reversable hash of the Discord webhooks along with the spreadsheet ID is stored to server logs to aid in tracing and control of nefarious actors.

Usage of this Add-On assumes full compliance with both Google Sheets Terms of Service as well as Discord's Terms of Service.
