# DiscouRSS - RSS to Discourse bridge via Google Sheets

DiscouRSS is a Google Sheets AppsScript that allows reads RSS Feeds (such as news, Letterboxd reviews, GoodReads activity) and pipe those events to a Discord channel, optionally attributing the post to a specific user.

The configuration and execution is handled entirely within Google Sheets, allowing for easy management with no cost (beyond a Google account).
